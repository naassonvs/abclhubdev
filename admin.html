ℹ️ Como funciona

Cada visita ao index.html é registrada automaticamente com data, hora e localização aproximada via IP (ip-api.com). Os dados ficam no localStorage do visitante — ou seja, contam visitas feitas neste navegador/dispositivo. Para analytics de todos os visitantes, recomenda-se integrar o Google Analytics ou Netlify Analytics no futuro.
